# Code Convention

This document outlines the coding conventions and best practices for this project. Following these guidelines ensures consistency, readability, and maintainability across the codebase.

## Principles

- **Reusable Utility Functions**: Prioritize writing reusable utility functions to avoid code duplication and improve maintainability.
- **Reusable View Functions**: Extract presentation logic into reusable view functions or components to keep templates clean and consistent.
- **Modular File Structure**: Break down large files into smaller, focused modules to enhance readability and simplify testing.
- **Documentation for Complexity**: Provide clear comments and documentation for complex functions or logic to aid fellow developers in understanding the codebase.
